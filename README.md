# Java Brains - Microservices

This project is based on YouTube Java Brains tutorial about Spring Cloud

## Installation

Use [Gradle build tool](https://gradle.org/guides/#getting-started) to install.

```bash
gradle build
```

## Usage

```python
java -jar build/
```
